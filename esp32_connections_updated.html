Master Connection Table: ESP32 + DHT11 + INA219

Component Pin Name Target Connection Description / Function
Main Power Supply
Power Supply Charger (+) vin+ (INA219) Routes the main 5.5V positive power into the sensor.
Power Supply Charger (-) GND (Breadboard) The common ground reference for the entire system.
ESP32 Board GND GND (Breadboard) Connects ESP32 ground to the common system ground. (CRITICAL for I2C logic reference)
INA219 (Load Path)
INA219 Load vin+ Charger (+) Receives the main positive power from the charger.
INA219 Load vin- V5 (ESP32 VIN) Outputs the measured 5.5V power to run the ESP32.
INA219 (I2C Logic)
INA219 Logic VCC 3V3 (ESP32) 3.3V logic power for the INA219 chip.
INA219 Logic GND GND (Breadboard) Common ground.
INA219 Logic SDA G21 (GPIO 21) I2C Data line.
INA219 Logic SCL G22 (GPIO 22) I2C Clock line.
DHT11 (Sensor)
DHT11 + (VCC) 3V3 (ESP32) 3.3V power for the temperature/humidity sensor.
DHT11 - (GND) GND (Breadboard) Common ground.
DHT11 out G4 (GPIO 4) Digital data signal for temp/humidity readings.
